## 📖 What This Is

**TON Price Predictor** is a data-driven forecasting tool for The Open Network (TON) blockchain's native cryptocurrency. Using historical price data, on-chain metrics, and market sentiment analysis, it generates short-term and mid-term price predictions. This is an experimental utility for research and educational purposes — not financial advice.

---

## ✨ Key Features

- **📊 Multi-Timeframe Forecasting** — Predictions for 1h, 24h, 7d, and 30d horizons
- **🔗 On-Chain Data Integration** — Analyzes TON blockchain activity (transactions, active wallets, staking)
- **🧠 ML-Powered Model** — Lightweight ensemble of LSTM + Random Forest trained on 2022–2025 data
- **📈 Visual Charts** — Interactive candlestick charts with prediction overlays (Plotly.js)
- **⚡ Real-Time Alerts** — Telegram/email notifications when prediction thresholds are triggered
- **🔄 Auto-Updating** — Fetches latest price & on-chain data every 15 minutes
- **🔒 No API Key Required** — Uses public TON Center and CoinGecko endpoints
- **🆓 Completely Free** — No subscriptions, no hidden paywalls

## 📦 Installation

### Prerequisites
- Python 3.9+ (for desktop version)
- Node.js 18+ (for web version)
- Git

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/ton-price-predictor-2026.git
   cd ton-price-predictor-2026
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt   # Python version
   # OR
   npm install                       # Web version
   ```

3. **Configure settings** (optional)
   ```bash
   cp config.example.yaml config.yaml
   # Edit config.yaml with your preferences
   ```

4. **Run the predictor**
   ```bash
   python predictor.py --mode live
   # OR
   npm start
   ```

5. **Open in browser** (web version)
   ```
   http://localhost:3000
   ```

## ❓ FAQ

### ❔ Is this safe to use in 2026?
Yes. The tool only reads public blockchain data and market APIs. It does not require access to your wallet, private keys, or exchange accounts. No risk of asset loss or account compromise.

### ❔ How often are predictions updated?
The model retrains automatically every 24 hours using the latest 90 days of data. Real-time predictions update every 15 minutes when the tool is running.

### ❔ Why is my prediction different from other sources?
Each model uses different parameters, data sources, and timeframes. Our model weights on-chain activity (TON blockchain metrics) more heavily than pure price history, which can produce divergent forecasts especially during network events.

### ❔ Can I use this for trading?
This tool is for **educational and research purposes only**. Cryptocurrency markets are highly volatile. Never trade based solely on automated predictions. Consult a financial advisor.
